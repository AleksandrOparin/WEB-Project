Разрабатываемое приложение представляет собой некоторый аналог Stack Overflow, обладающий следующим функционалом: добавление вопроса, добавлением тэгов к вопросам, сортировкой вопросов по тэгу, сортировкой по лайкам, добавлением ответов на вопросы и выбор верного ответа.

## Установка для Windows

Для запуска Вам понадобятся _Python_. Все остальные нужные пакеты представлены в файле _requirements.txt_.

1. Для начала склонируйте проект, прописав в терминале команду:
   ```
   git clone https://github.com/AleksandrOparin/WEB-Project.git
   ```

2. Проверьте, что у вас установлен venv следующей командой:
   ```
   virtualenv --version
   ```

Если его нет, то установите.
   ```
   pip install virtualenv
   ```

3. Находясь в корневом каталоге проекта, пропишите в терминале в зависимости от терминала следующее:

В PowerShell:
   ```
   python -m venv venv
   venv\Scripts\Activate.ps1
   pip install -r requirements.txt
   ```

В cmd.exe:
   ```
   python -m venv venv
   venv\Scripts\activate.bat
   pip install -r requirements.txt
   ```

В wsl:
   ```
   virtualenv -p python3 venv
   source venv/bin/activate
   pip install -r requirements.txt
   ```

Выйти из виртуального пространства поможет команда:
   ```
   deactivate
   ```

Таким образом мы перешли в виртуальное пространство и скачали скачать все необходимые зависимости для запуска проекта.

## Генерация данных

> # Все команды выполняются в venv!

1. Выполните миграцию командой в корневом каталоге:
   ```
   python manage.py migrate
   ```

2. Поднимите СУБД postgresql. Делается это командой в терминале из корневого каталога:
   ```
   docker-compose up -d --build
   ```

Опция -d запускает СУБД postgresql в фоновом режиме/

Если понадобиться завершить, то пропишите команду:
    ```
   docker-compose down
   ```

3. Сгенерируйте необходимое число данных. Для этого также в корневом каталоге пропишите:
   ```
   python manage.py fill_db [--ratio number]
   ```

Где number — коэффициент заполнения сущностей. По умолчанию number = 10000, что может быть много. В соответствии с этим параметров формируются следующее число данных:

* число пользователей = ratio;
* число вопросов = ratio * 10;
* число ответов = ratio * 100;
* число тэгов = ratio;
* число оценок пользователей = ratio * 200.

## Запуск

Находясь в корневом каталоге проекта внутри venv, пропишите в терминале следующее:
```
python manage.py runserver
```

После перейдите в браузере на localhost (http://127.0.0.1:8000/).
